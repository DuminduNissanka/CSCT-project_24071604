# CSCT-project_24071604

# Repository contains: 

## datasets Zip file
        * bristol_EV_Stations.csv - contains Bristol city EV charging points
        * synthetic_EV_sessions_bristol.csv - synthetically generated EV sessions
        * EV_sessions_USA.csv - contains USA based EV sessions for mapping to Bristol stations
        * bristol_weather_2024.csv - contains real Bristol weather data in 2024
        * BRISTOL_TRAFFIC_DATA.csv - contains traffic data for Bristol
        * bristol_traffic_average_day.csv - contains average traffic per day in Bristol
        * FinalDataSet.csv - combined final dataset.

## Data_cleaning.ipynb - code to clean data
## SyntheticDataGeneration.ipynb - data simulation for EV sessions
## BristolWeatherAPI.ipynb - code to get weather data from API
## Weather_data_merging.ipynb - code to merge weather data to main dataset
## TrafficDataMerging.ipynb - code to merge traffic data to main dataset
## EDA01.ipynb - code to EDA including univariate analysis
## EDA02.ipynb - code to EDA including bivariate, multivariate and descriptive analysis
## ML_models.ipynb - code to all ML model training, results and evaluation
